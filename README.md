# Corlius' Tab

欢迎使用 Corlius' Tab，一个干净的新标签页。

## 功能

1. **深浅色模式适配**：
   - 根据系统主题自动切换浅色或深色模式。
   - 可分别为浅色和深色模式设置不同的壁纸。

2. **搜索引擎选择**：
   - 可选Google、Bing和Baidu等搜索引擎。

## 使用方法

1. **安装插件**：

   [Custom New Tab Page](https://chromewebstore.google.com/detail/custom-new-tab-page/bfhappcgfmpmlbmgbgmjjlihddgkeomd)

2. **设置 URL**：

   - 在插件中设置 URL 为：https://tab.corlius.top/ 。

## 自定义设置

### 搜索引擎选择

1. 点击右下角的设置图标打开设置菜单。
2. 通过下拉菜单选择搜索引擎。
3. 通过搜索框进行搜索。

### 壁纸设置

1. 点击右下角的设置图标打开设置菜单。
2. 通过下拉菜单选择浅色和深色壁纸。
3. 选择的壁纸会自动保存并应用。

## 技术细节

- **本地存储**：保存您的壁纸和搜索引擎偏好。
- **动态主题切换**：自动适应系统的浅色或深色主题。
- **响应式设计**：适用于桌面和移动设备。
